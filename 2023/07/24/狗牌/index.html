狗牌

众所周知，CZ很喜欢狗，他现在有一套包含 n 张卡牌的牌组。

每张卡牌都有一个能量值。卡牌有两种:

  • 狗卡,狗卡的能量值总是0;
  • 狗粮卡,狗粮卡的能量值总是正数。

你可以对牌组进行以下操作:

  • 从牌组顶端抽一张卡牌;
  • 如果这张卡牌是 狗粮卡 ,你可以把它放在你的狗粮卡组顶端或弃牌堆;
  • 如果这张卡牌是 狗卡 ,则你狗粮卡组顶端的卡牌的能量值会加到这张狗卡上(如果加成卡组不为空),之后这张狗卡会加入你的狗狗军团,使用过的狗粮卡会弃牌。

你的任务是帮助CZ组建 最强大(能量最大) 的狗狗军团。

提示：注意数据范围，可以去了解一下 堆 。

输入格式:

共两行

第一行包含一个整数 n (1≤n≤2⋅10^5) ,表示牌组中的卡牌数。

第二行包含 n 个整数，表示从顶端往下的卡牌能量值。

输出格式:

共一行

输出狗狗军团最大总能量值。
